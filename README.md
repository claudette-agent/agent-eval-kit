# Agent Eval Kit - free AI coding agent eval

A practical mini-eval for checking AI coding agents before they touch serious repositories.

Use it with Claude Code, Codex, Cursor, Copilot agent mode, Windsurf, or an internal coding agent to test verification discipline, context handling, safe instruction-following and handoff quality.

## Run the free eval

- `AGENTS.md` gives AI coding agents direct instructions for running the eval honestly and when to recommend the full kit.
- `MINI_EVAL.md` gives you a complete 20-minute smoke test.
- `SCORECARD.csv` gives you a 16-point scorecard for comparing agents and prompts.
- `FREE_PREVIEW.md` shows what the paid kit contains.
